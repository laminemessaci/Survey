## Installation

To run this project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/survey-agency.git
   cd survey-agency
  
2. Install dependencies:

   ```bash
   npm i

3. Start development server:

    ```bash
    npm start
